2.      What is the difference between a method and function?

Functions and methods are the same in JavaScript, but a method is a function, which is a property of an object. A function can be called directly by its name but methods are called by its object name and its method name using dot notation or square bracket notation.

10.  Write a Javascript function that takes any number of input arguments and returns the product of the arguments.

function productOfArguments() {

let i;

let productResult = 1;

for (i = 0; i < arguments.length; ++i) {
